# Stats(Estadísticas).
El paquete Stats (Estadísticas) de V12 es el módulo encargado de la recopilación y administración de estadísticas dentro del ERP, ofreciendo herramientas para consolidar información de diferentes procesos y transformarla en datos útiles para la toma de decisiones. Su propósito es centralizar métricas clave y facilitar el análisis de desempeño en áreas operativas, financieras y administrativas.
